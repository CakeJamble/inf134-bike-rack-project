# inf134-bike-rack-project

## Get Started

1. Clone the repo
2. Make sure you have `npm` installed
3. `cd` into `bike-rack-proj` and run the command `npm install`
4. Once finished, run the dev container with `npm run dev` to see changes in your browser
5. Start making pages or changes in `src/`. Use commands to generate template pages, etc.

If you need to run other commands, check out `package.json` for other options, such as

```json
  "scripts": {
    "dev": "next dev --turbopack",
    "build": "next build",
    "start": "next start",
    "lint": "next lint"
  },
```
